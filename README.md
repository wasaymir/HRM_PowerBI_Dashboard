# HR Analytics Power BI Dashboard

A complete HR Analytics Dashboard built in Power BI covering:

- Workforce demographics & map by country  
- Satisfaction & performance analysis  
- Compensation insights & income distribution  
- Attrition trends & tenure analysis  
- Dynamic filters & interactive navigation

## Files Included
- `.pbix` Power BI dashboard file  
- Screenshots of dashboard pages  

## Skills Used
- Power BI, DAX, Data Modeling, Data Visualization, UX Design

## How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Refresh data if needed
